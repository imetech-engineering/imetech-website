POC Proximity Chat

A proof-of-concept Ultra-Wideband (UWB) communication system with real-time distance measurement and simulated audio interaction. Demonstration of proximity-based audio concepts with ESP32-UWB units and Raspberry Pi hub.

October 2025

Project Description

The POC Proximity Chat project was developed by IMeTech Engineering as a proof-of-concept for an interactive communication system based on Ultra-Wideband (UWB) technology. The goal is to demonstrate how Qorvo DW3000 modules can be used to reliably measure distances between people and translate this information into simulated audio interaction.

The system consists of portable ESP32-UWB units and one central hub based on a Raspberry Pi. The UWB units communicate with each other for accurate distance measurement and send the measurement results via Wi-Fi to the hub, which visualizes the data in real-time through a web interface. The interface shows connected devices, distance measurements and simulated volumes, making the operation of proximity-based audio concepts comprehensible.

This proof-of-concept forms the technical basis for further expansion with audio hardware, custom PCBs and integration into large-scale event applications. The project demonstrates the possibilities of UWB technology, real-time communication and embedded systems for innovative event experiences.
