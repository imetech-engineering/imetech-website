Project

Rollator Control

Innovative mobility assistance unit with integrated laser projection for improved walking confidence.

February 2026
PCB Design Embedded Systems Medical Tech Small Series Production

The Rollator Control Unit was developed as a specialized mobility aid component, specifically designed for users living with Parkinson’s. A core feature of this system is the high-precision laser projection, which casts a visual guide line on the ground when the user stands still. This visual stimulus helps break "freezing" episodes and provides a clear reference point for direction and movement initiation.

IMeTech Engineering managed the complete project lifecycle: from initial PCB design and firmware development (ESP32-C3 based) to small-series manufacturing. The unit features intelligent motion detection, precise battery analytics, and ultra-low-power sleep modes. Integrated USB-C charging ensures the system is modern, durable, and easy for end-users to maintain.

This project showcases a meaningful application of assistive technology, proving how custom electronics can directly improve life quality through smart sensing and ergonomic design.
