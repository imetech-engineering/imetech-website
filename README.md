# IMeTech Website

This is the source code for the IMeTech Engineering website.

## Structure
- `index.html` — Main landing page
- `style.css` — Site-wide styles
- `script.js` — JavaScript for interactivity
- `images/` — Site images and assets
- `algemenevoorwaarden/` — Terms & conditions (HTML)
- `disclaimer/` — Disclaimer (HTML)
- `bestanden/` — Downloadable PDFs

## Usage
This is a static website. To view locally, open `index.html` in your browser.

## Deployment
Simply upload the contents to your web server or use GitHub Pages.
